<h1 align="center">Tutorly Chat Feature</h1>

<p align="center">
  A Chat feature that allows any two Tutorly Users to communincate with each other via text.
</p>

<p align="center">
  This feature is implemented with Firebase Spark Plan. Future updates will include a self-hosted solution.
</p>

Tutorly Chat Feature is a platform for creating in-app chat experiences using Flutter. This feature folder Implements its backend using Firebase.

Version 1.0.0 Features:
- [x] Chat with any Tutorly User
- [x] Send and Receive Messages
- [x] View Chat History
- [x] Delete Chat History
- [x] Search any User to Chat with

Goals for Version 1.1.0:
- [x] Implement Firebase Cloud Messaging for Notifications
- [x] Chats Room does not Update with Latest room changes.
- [x] Implement Chat Room Search
- [x] Janky Loading when loading a chat room
- [x] Have Button to lead to User Profile

### Requirements

`Dart >=2.19.0` and `Flutter >=3.0.0`, [Firebase](https://firebase.google.com) project.
